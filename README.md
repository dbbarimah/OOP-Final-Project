# OOP-Final-Project
The Smart Carbon Emissions Tracker

This is a simple Java Application which uses OOP principles
to calculate the carbon emissions of a household over a
specified period (ie. a week).
It uses their food consumption, transportation, electricity 
usage and many more to find the total emissions of the household
for the week.
